• Description

    One of my projects is this very website you are currently browsing through.

    Technology used

    The website is written in HTML and CSS for the formatting. JavaScript is used for the functionality (like the menu animations for example).

    The website is hosted on a VPS by Vultr, the domain name is provided by Namecheap.

    I host using nginx on a Debian server, the backend is written in Flask and thus I use gunicorn to run the website.

    I did everything myself, however the HTML, CSS, and JavaScript are adjusted from a template by Nicola Tolin, which you can read about in my credits section.
